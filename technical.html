Hosting details

The domain name yaqinhasan.com is hosted on AWS Route 53 name servers.
The website is being hosted on an OVH VPS, running Ubuntu 22 LTS.
The webserver is Apache2 http server.

SSL is provided through a free OVH SSL gateway, sits between the domain name and the server. Though I do want to eventually switch over to using certbot, because it would allow me to use apache vhosts. I want to eventually host multiple sites on this VPS, not just this personal site.
